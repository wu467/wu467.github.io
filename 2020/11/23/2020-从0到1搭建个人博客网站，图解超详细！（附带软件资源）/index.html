2020 从0到1搭建个人博客网站。

发表于 2020-11-23 分类于 博文

前言

博主趁着放假的两天时间，利用网上大佬们建站的个人经验博文，也从零基础成功的搭建了一个基于Github + Hexo的个人博客网站。特写一篇博文记录下来！

一、准备工作

1.1 入门知识

个人博客网站 =域名 + 空间
想要搭建个人博客网站，域名和空间是必不可少的。举个栗子来说明一下吧，域名就相当你家的地址，空间(服务器)相当于你的家。想要访问你家就必需先知道你家的地址，再通过地址访问你家。

  • 域名。 这个玩意看个人需求，有贵的也有便宜的，但用起来没什么差别。你可以在腾讯云或阿里云上购买注册，我就是在腾讯云上花了七块钱弄的，我的域名：www.wwqi.top欢迎大家访问 :)
  • 空间。个人理解就相当于一个存储空间，有了它，自己写的文章才有存放的空间。这个有免费的也有收费的，收费的当然要好一些，但对于我这个贫穷大学生来说免费的当然是第一选择啦（白嫖使我快乐，hhh）。免费的也就是Github了，既免费又稳定，缺点就是国内访问速度慢点，但如果你挂了梯子当我没说。其实也可以考虑下国内的码云，作用和GitHub一样，也很不错。两者都可以上传自己的网站项目并定义自己的域名。

1.2 所需软件

Git和Node.js

  • Git 作用：在本地搭建好个人博客网站项目后，通过该软件与Github产生关联，将其上传到Github上，然后我们就可以通过域名来访问我们的网站啦。后期维护和更新blog也会用到。
  • Node.js作用：这个捏我也不大清楚，貌似就是提高博客网站的响应速度，先安装就完事了。

博主只是提到了这两个软件在搭建博客中的作用，其实这两个软件功能非常强大，作用也远远不止这么多。
